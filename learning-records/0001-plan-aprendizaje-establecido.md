# Plan de aprendizaje de IA establecido desde cero

El usuario llegó con un plan de aprendizaje estructurado en 9 fases (Fase 0 — Fundamentos Conceptuales hasta Fase 8 — IA Aplicada a Otras Áreas). Esto indica que ya tiene claridad de alto nivel sobre el ecosistema IA y puede nombrar las áreas clave (LLMs, prompt engineering, agentes, RAG, producción). El punto de partida es la Fase 0 — no se asume ningún conocimiento técnico previo de LLMs.

**Implications**: Las primeras lecciones deben construir fundamentos sólidos antes de avanzar a prompt engineering. No re-enseñar conceptos de alto nivel que el usuario ya conoce por nombre.
