# LLM Learning Path

Un camino de aprendizaje autodirigido desde usuario casual de IA hasta ingeniero de LLMs en producción. 55 lecciones HTML interactivas estructuradas en 9 fases: ingeniería de prompts, integración con IDEs, construcción de APIs, agentes autónomos, RAG y sistemas de IA en producción.

## 📚 Contenido del Curso

Este curso está estructurado en 9 fases progresivas:

- **Fase 0 - Fundamentos Conceptuales**: Conceptos básicos de LLMs, tokens, contexto, temperatura, embeddings, limitaciones y alucinaciones
- **Fase 1 - Prompt Engineering**: Anatomía de buenos prompts, técnicas básicas y avanzadas, gestión de contexto, evaluación y refinamiento
- **Fase 2 - IDE Integration**: Integración con herramientas como Cursor y Copilot, autocompletado, refactorización, code review
- **Fase 3 - API Building**: Primeros pasos con APIs, prompt caching, tool use y function calling, structured outputs, streaming
- **Fase 4 - Autonomous Agents**: Conversaciones con memoria, RAG, embeddings, búsqueda semántica, diseño de herramientas y agentes
- **Fase 5 - Multi-agent Systems**: Bucles de agentes, arquitectura de agentes, frameworks como LangChain y CrewAI
- **Fase 6 - Production**: Fine-tuning vs prompting, evaluación sistemática, observabilidad, logging, seguridad
- **Fase 7 - Optimization**: Escalado, optimización de costes, patrones de arquitectura
- **Fase 8 - AI Applications**: Casos de uso en investigación, automatización de flujos de negocio y otras aplicaciones
